# 前端细节优化清单（第一版）

目标：在不改动核心业务流程的前提下，提升报告表格与趋势图的可读性、操作一致性和视觉层级。

## 1) 问题与优化点

### P0（立即优化）

1. 异常值提示不够聚焦  
   - 现状：异常值主要靠红色文字，长表格下不够醒目。  
   - 优化：异常值增加浅红背景与圆角，保留红字+箭头。

2. 删除操作颜色与“异常红色”冲突  
   - 现状：删除按钮使用红色，容易和异常值语义混淆。  
   - 优化：删除按钮改为品牌蓝，hover 使用浅蓝背景。

3. 表格横向浏览时定位困难  
   - 现状：列较多时横向滚动后，日期列容易离开视野。  
   - 优化：日期列改为左侧 sticky，便于对齐阅读。

4. 数值列对齐不统一  
   - 现状：数值、单位、箭头混排，扫描速度慢。  
   - 优化：数值列右对齐；单位弱化显示，异常值保持视觉优先级。

### P1（本轮可做）

5. 趋势图 tooltip 精度不统一  
   - 现状：部分指标小数位显示不一致。  
   - 优化：按指标控制显示精度（如红细胞 2 位、其余常规 1~2 位）。

6. 参考范围可见性优化  
   - 现状：部分情况下参考带较淡。  
   - 优化：轻微提高参考带对比度，并保持暗色主题可读。

## 2) 本次落地范围（第一批）

- [x] 异常值增加浅红底高亮  
- [x] 删除按钮改为蓝色语义  
- [x] 表格日期列 sticky  
- [x] 数值列右对齐（保留日期/操作列左对齐）

## 3) 验收标准

1. 表格中任意超出参考范围数据，均出现红字+浅红底+箭头。  
2. 删除按钮不再使用红色，且 hover 有清晰反馈。  
3. 表格横向滚动时，日期列始终可见。  
4. 数值列视觉上右对齐，便于纵向比对。  

## 4) 后续建议（第二批）

- 图表 tooltip 精度分级策略（按指标类型自动格式化）。  
- 增加“仅看异常”过滤开关。  
- 添加“最近一次 vs 上一次”变化幅度提示。  
- 导出当前表格为 CSV（用于医生沟通/病历归档）。
